# Website Landing Page - RSUD Bolaang Mongondow Timur

Proyek ini adalah landing page resmi untuk **Rumah Sakit Umum Daerah (RSUD) Bolaang Mongondow Timur**, yang dirancang untuk memberikan informasi kepada masyarakat mengenai layanan rumah sakit, profil, kontak, dan update terbaru dari RSUD Boltim.
